# Cat API

A simple web app that displays random cat images from the internet.

## Features

- Displays a random cat image on page load
- Refresh button to load a new cat image
- Error handling for failed API requests
- Clean, simple interface

## How It Works

The app fetches random cat images from [The Cat API](https://thecatapi.com/) and displays them on the page. Click the "Refresh For a New Picture" button to get another cat.

## Usage

1. Open `index.html` in your web browser
2. View a random cat image
3. Click the refresh button to load a new image

## Technologies

- HTML
- JavaScript (async/await, Fetch API)
- The Cat API
